# block-sample
ブロック崩し的なゲーム

## ビルド方法

### Node.jsの環境を設定し、node、npmコマンドを使えるようにする
- node.js
- npm


[Node.js公式サイト](https://nodejs.org/ja/)

### node_moduleを取り込む

```
npm install
```

### 実行する

```
npm run dev
```
